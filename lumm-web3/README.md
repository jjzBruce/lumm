## 搭建本地区块链并导入测试账号

### ganache

> ganache v7.9.1 (@ganache/cli: 0.10.1, @ganache/core: 0.10.1)

```js
// 安装
npm i ganache -g
// 启动
ganache -d
// 会产生10个测试钱包账号，复制私钥导入到小狐狸钱包中。
// 导入完之后发现账号下面的以太坊币是0。
// 此时需要在小狐狸中添加网络，将本地的区块链网络添加进去。具体参数在 ganache 启动所在终端中已经打印。
```
